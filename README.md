# Retail ERP System

Warehouse & Store Management ERP System built with Flask and SQLite.

This project is a mini ERP platform designed for supermarket and retail store operations.  
It includes inventory management, warehouse stock control, order workflows, product management, and approval processes.

-----------------------------------------------------------------------------------------------------------------------

## Features
----------------------------------------

1. Authentication & Roles
- User login system
- Role-based access control
- Warehouse users
- Store Managers
- Assistant Managers

2. Warehouse Management
- Warehouse inventory monitoring
- Low stock alerts
- Product activation / deactivation
- Dynamic category management
- Product search & filtering

3. Orders Workflow
- Store order creation
- Pending approval workflow
- Manager approval process
- Automatic warehouse stock reduction
- Stock validation before order submission
- Cart management system

4. User Experience
- Flash notifications
- Responsive dashboard layout
- Styled UI with custom CSS
- Interactive forms
- Dynamic filtering


## Technologies Used
----------------------------------------

- Python
- Flask
- SQLite
- HTML5
- CSS3
- JavaScript
- Jinja2 Templates


## Database Structure
----------------------------------------

Main database tables:
- users
- stores
- store_stock
- products
- warehouse_stock
- orders
- order_items
- categories


## Installation
----------------------------------------
Clone the repository:


```bash
git clone https://github.com/GeorgeKat039/retail-erp-system.git
```

Navigate into the project folder:

```bash
cd retail-erp-system
```

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```
